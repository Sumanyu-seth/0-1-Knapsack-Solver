Project Overview

This repository provides an efficient solution for the 0/1 Knapsack Problem using the Dynamic Programming (DP) approach. The goal is to maximize the total value of items selected without exceeding a set weight capacity.

Optimal Result: Guarantees the globally maximum profit using bottom-up DP tabulation.
